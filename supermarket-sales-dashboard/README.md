# 🛒 Supermarket Sales Dashboard

## 📊 Overview

This project presents a comprehensive analysis of supermarket sales data using Power BI.
It provides insights into customer behavior, product performance, and overall business trends.

---

## 🚀 Key Features

* Sales analysis by **Month, Branch, and Product Line**
* Customer segmentation (**Gender & Customer Type**)
* Payment method analysis
* Daily sales trend with **average benchmark line**
* Customer satisfaction analysis using **Ratings**
* **Average Order Value (AOV)** calculation

---

## 📈 Key Insights

* 🟢 **Branch C** achieved the highest sales performance
* 🔴 **February** recorded the lowest sales
* 🟡 **Food & Beverages** is the top-performing category
* 👥 Members and female customers spend more on average
* 📉 Sales show noticeable daily fluctuations
